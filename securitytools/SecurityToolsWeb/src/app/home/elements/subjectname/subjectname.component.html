Common Name (CN) Your name or your server hostname. Might be a fully qualified domain name (FQDN). Organizational Unit name (OU) Department within the organization. Organization Name (O) Company's legally registered name. State or Province name (ST or SP or S) The state/province where your company is legally located. CountryName (C) Two-letter country code (ISO 3166). Domain Component (DC) Identifies the domain components of a domain.
